Intro to Programming Notes Important Concepts

The World Wibe Web is a collection of documents, most are made up of HTML.
HTML stands for Hypertext Markup Language.
The browser reads, interprets, and displays these documents.
Servers are computers that host files that make up the web.
Computers make requests via the internet to servers, the internet and servers communicate through http.
HTTP is the main protocol of the web.

Coding is when programmers write text in a language computers understand, so the computer can follow the instructions.
Computers are stupid, even a small mistake like a typo can break the whole program.
There are too many details to remember everything, so programmers can just look something up if they forget how to do it.

Basic HTML vocabulary:
Tag - HTML tags are always contained in angle brackets < >. Most tags have opening < > and closing < / > tags, but some tags called void tags do not require a closing tag.
Element - An HTML element refers to everything within a set of opening and closing tags.
Attribute - This is a property of an HTML element, such as the url in < a href = "url" >

Misc notes
There are inline tags and block tags - block tags create an invisible box or container around the content that has a height and width, while inline tags do something within the line.
In HTML all "whitespace" turns into a single space. To make a new line render you'd need to use the break tag or the paragraph tag.
HTML documents are made up of text content (what you see), markup (what it looks like), references to other documents (such as images or videos), and links to other pages.
